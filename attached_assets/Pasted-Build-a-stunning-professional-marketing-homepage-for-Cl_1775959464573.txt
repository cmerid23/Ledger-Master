Build a stunning, professional marketing homepage for ClearLedger — a modern accounting SaaS. This is the public-facing landing page at the / route. It should feel like a premium product, comparable to QuickBooks or Wave but more modern and sleek. Do NOT redirect to login from the root — show this homepage instead. Only redirect to dashboard if the user is already logged in.

TECH

Single page React component: client/src/pages/HomePage.jsx
Add route in App.js: <Route path="/" element={<HomePage />} />

Use Framer Motion for scroll animations (install if not present)
